アカウント・データ削除のご案内

Class Orange（みまもり / はじまりの刻）
最終更新日：2026年6月28日

Class Orange が提供するアプリについて、アカウントおよび関連データの削除方法をご案内します。 アプリの種類によって手順が異なります。

みまもり（家族アカウント）の削除

みまもりアプリはメールアドレスでログインするアカウントがあります。 アカウントの削除は、アプリ内から、ご自身で行っていただけます。

  1. みまもりアプリを開き、「設定」タブを開く
  2. 「アカウントを削除」を選ぶ
  3. 確認画面で削除を実行する
  4. 削除が完了すると自動的にログアウトされます

アプリを開けない場合や、その他のご事情で削除を希望される場合は、下記の窓口へメールでご連絡ください。 ご本人確認のため、ご登録に使用したメールアドレスからお送りください。原則として30日以内に対応します。

  • 連絡先：support@class-orange.com

削除されるデータ（みまもりアカウント）

  • アカウント情報（メールアドレス等の登録情報）
  • プッシュ通知トークン
  • 見守りの紐付け情報（どの端末を見守っているかの設定）
  • 発行済みの招待情報

削除されないデータ

見守られる側の方（おかえりナビ・はじまりの刻をお使いの方）の端末に紐づく記録は、 その方ご本人のデータであるため、見守る側のアカウントを削除しても保持されます。 見守られる側のデータ削除をご希望の場合は、下記「端末ベースのアプリ」の手順をご参照ください。

なお、アカウントを削除しても、見守り対象ご本人の端末に記録された活動履歴は本人のデータとして残ります。 再度招待すると、その履歴は再び表示されます。

はじまりの刻・おかえりナビ（端末ベース・アカウントなし）

これらのアプリはログイン不要で、端末そのものが識別情報となります。アカウントはありません。

  • アプリをアンインストールすると、端末内に保存された識別情報は削除されます。
  • サーバーに保存された記録（アラームログ等）の削除をご希望の場合は、 support@class-orange.com へご連絡ください。

一定期間後に自動削除されるデータ

アラームログなどの活動記録は、削除リクエストの有無にかかわらず、 90日間の保持期間を過ぎたものから自動的に削除されます。

ご注意

アカウントを削除すると、見守りの履歴や設定はすべて失われ、元に戻すことはできません。 また、ご家族（見守る側・見守られる側）との連携も解除されます。
