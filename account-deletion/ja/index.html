アカウント・データ削除のご案内

Class Orange（みまもり / はじまりの刻）
最終更新日：2026年6月28日

Class Orange が提供するアプリ（「みまもり」「はじまりの刻」）について、 アカウントおよび関連データの削除をご希望の方は、このページの手順に従ってリクエストしてください。

削除をリクエストする方法

以下のメールアドレスまで、件名に「アカウント削除希望」と記載のうえ、ご連絡ください。 ご本人確認のため、アプリへのご登録に使用したメールアドレスからお送りください。

  • 連絡先：support@class-orange.com

ご連絡を受け取り次第、原則として30日以内にアカウントおよび関連データを削除します。

削除されるデータ

  • アカウント情報（メールアドレス等の登録情報）
  • プッシュ通知トークン
  • 見守りの設定・紐付け情報
  • 見守り対象者のアラームログ・アラーム設定（みまもり）
  • アラーム設定・アラーム確認ログ（はじまりの刻）

一定期間後に自動削除されるデータ

アラームログなどの活動記録は、削除リクエストの有無にかかわらず、 90日間の保持期間を過ぎたものから自動的に削除されます。

ご注意

アカウントを削除すると、見守りの履歴や設定はすべて失われ、元に戻すことはできません。 また、ご家族（見守る側・見守られる側）との連携も解除されます。
